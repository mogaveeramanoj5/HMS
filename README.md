Hospital Management System (HMS)

A web-based Hospital Management System developed using PHP, HTML, CSS, JavaScript, and MySQL. This project helps manage hospital operations such as patient records, doctor details, appointments, billing, and admin activities efficiently.

Features:-
  Patient Registration and Management
  Doctor Management
  Appointment Booking System
  Admin Dashboard
  Secure Login System
  CRUD Operations
  Responsive User Interface
  Database Connectivity using MySQL
  
Technologies Used:-
  Frontend: HTML, CSS.
  Backend: PHP
  Database: MySQL
  Tools: VS Code, XAMPP
  
Project Modules:-
  Admin Module
  Doctor Module
  Patient Module
  
Installation Steps:-
  Install XAMPP
  Copy the project folder into htdocs
  Start Apache and MySQL in XAMPP
  Import the SQL database file into phpMyAdmin
  Open browser and run:
  http://localhost/hms
  
Database:-
MySQL database used for storing:
  Patient details
  Doctor records
  Appointment data
  Admin 
  Medical report
